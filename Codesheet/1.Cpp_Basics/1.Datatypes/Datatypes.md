# Datatypes in C++

The most important datatypes in C++ are:
|  **Datatype**  |     **Bytes**     |      **Minimum Value**      |       **Maximum Value**       | 
|----------------|-------------------|-----------------------------|-------------------------------|
| int            | 4                 | -2,147,483,648              |   2,147,483,647               |
| Unsigned int   | 4                 |  0                          |   4,294,967,295               |
| Unsigned short | 2                 |  0                          |   65,535                      |
| Unsigned long  | 8                 |  0                          |   18,446,744,073,709,551,615  |
| Long long      | 8                 | -9,223,372,036,854,775,808  |   9,223,372,036,854,775,807   |
| Float          | 4                 |  1.17549e-38                |   3.40282e+38                 |
| Double         | 8                 |  2.22507e-308               |   1.79769e+308                |
| Long double    | 12                |  3.3621e-4932               |   1.18973e+4932               |
| Char           | 1                 | -128                        |   128                         |
| String         | Depends on Input  |                             |                               |
| Bool           | 1                 |  0                          |   1                           |

You can run the following code and uncomment the datatypes you want to run.
```cpp
#include<bits/stdc++.h>
using namespace std;

int main() {
    // int
    // int x = 10; // Declaring and initializing variable x
    // long
    // long y = 5000000000;
    // long long
    // long long z = 1000000000000000000;
    // short
    // short a = 30;
    // float
    // float b = 3.14;
    // double
    // double c = 3.14159265358979323846;
    // string
    // string d = "Hello World!";
    // char
    // char e = 'A';
    // getline
    // string f;
    // getline(cin, f);
    // cout << f;
    // bool
    // bool g = true;

    return 0;
}
```
